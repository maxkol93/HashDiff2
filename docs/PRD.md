# PRD: HashDiff — File Comparison Utility

## Overview
HashDiff — простая Windows desktop утилита для сравнения файлов и архивов по MD5 хешу. Цель — моментально визуально определить, идентичны ли файлы.

## Problem Statement
При работе с файлами, бэкапами и архивами регулярно возникает вопрос: «Это тот же файл или нет?». Ручное сравнение хешей неудобно — нужно открывать консоль, вводить команды, копировать и сравнивать строки. HashDiff убирает весь этот friction.

## Target Users
- Разработчики (проверка артефактов сборки)
- Системные администраторы (проверка целостности бэкапов)
- Технические пользователи (сравнение скачанных файлов)

## Core Value Proposition
Перетащи файлы → получи ответ за секунды. Зелёный = одинаковые, красный = разные.

## MVP Features

### F1: Dual Panel Layout
Два параллельных панели (Левая / Правая) для загрузки файлов.

### F2: Drag & Drop
Drag & drop файлов в каждую панель. Также кнопка «Открыть файл» как fallback.

### F3: MD5 Hashing
- Обычные файлы: MD5 от содержимого файла
- Архивы (.zip, .rar, .7z, .tar, .gz): MD5 от самого файла архива (как binary)

### F4: Visual Comparison
- Одинаковые файлы (MD5 совпадает) → строка подсвечена зелёным
- Разные файлы (MD5 не совпадает) → строка подсвечена красным
- Файл без пары → серый (ожидание)

### F5: Hash Display
Показывать MD5 хеш для каждого файла. При наведении — полный путь к файлу.

### F6: Multi-file Support
Можно загрузить несколько файлов в каждую панель. Совпадения выравниваются по строкам (matching по имени файла, fallback — по порядку).

## Out of Scope (MVP)
- Сравнение содержимого архивов (только файл как целое)
- SHA256 / другие алгоритмы
- Рекурсивный обход папок
- Экспорт отчётов
- История сравнений

## Success Criteria
- Пользователь загружает 2 файла и видит результат за < 2 секунд
- Приложение работает как единый `.exe` без установки
- Поддерживает файлы размером до 4 GB (streaming hash)
- Корректно отображает кириллические имена файлов

## Technical Constraints
- Platform: Windows 10/11 x64
- Runtime: Python 3.11+, упаковка PyInstaller
- GUI: Tkinter (встроен в Python, нет внешних зависимостей для GUI)
- Единый `.exe` файл на выходе
